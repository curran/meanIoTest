# meanIoTest
A test of mean.io.

```
sudo npm install -g mean-cli
cd repos/
mkdir meanIoTest
cd meanIoTest/
mean init myBlog
cd myBlog
npm install
bower install
npm install -g gulp
gulp
```

http://localhost:3000

http://learn.mean.io/ Mean starter documentation

To see the directory tree of the generated project:

`tree -I "node_modules|bower_components"`
